Fyrestone

Fyrestone is a content mod for the popular game Minecraft. It's written using the community-run Forge Modding API, entirely in Java.
It doesn't always follow best Java practices, due to the requirements of the Modding API, but I'm quite proud of how extensible its systems are.

Pxls

Pxls is an online collaborative art project, inspired by Reddit's 2017 April Fools event; /r/place.
It was written in collaboration with a friend, and is currently being maintained by various members of the community.
The backend is almost entirely Java, based on the Undertow Java Webserver, with some small parts of the code in Kotlin. The frontend is vanilla JavaScript.

Canvity

Canvity is a WIP TypeScript framework based on the OBS (Object-Behaviour-Scene) architecture of Unity.
It can be used both for games and general apps, and is designed to act as a useful layer of abstraction between the programmer and the HTML5 canvas.
